结构伪类选择器

根据元素的结构关系查找元素

结构伪类选择器是根据元素的结构关系来查找元素的选择器。它们可以帮助我们选择特定位置或特定状态的元素。

E:first-child -- 查找第一个E元素

E:last-child -- 查找最后一个E元素

E:nth-child(n) -- 查找第n个E元素（第一个元素N值为1）

:nth-child( 公式 )

根据元素的结构关系查找多个元素

nth-child(2n) -- 查找偶数位置的元素

nth-child(2n+1) 或 nth-child(2n-1) -- 查找奇数位置的元素

nth-child(5n) -- 查找5的倍数位置的元素

nth-child(n + 5) -- 查找第5个以后的标签

nth-child(-n + 5) -- 查找第5个以前的标签
